📱 习惯打卡 隐私政策

最后更新时间：2026年3月20日  |  版本：1.0.0

🔒 核心承诺：「习惯打卡」所有数据仅存储在您的设备本地，我们不收集、不上传、不分析您的任何个人信息。

一、我们收集哪些信息

「习惯打卡」是一款完全本地化的应用，不收集任何个人信息。具体说明如下：

数据类型是否收集说明
姓名、手机号、邮箱❌ 不收集无需注册或登录账号
位置信息❌ 不收集应用不请求位置权限
习惯数据✅ 本地存储仅保存在您的设备上，不上传云端
设备标识符❌ 不收集不读取设备 IMEI 或广告 ID
使用行为数据❌ 不收集不进行任何行为追踪或分析

二、我们如何使用您的信息

由于我们不收集您的个人信息，因此不存在对您数据的任何使用行为。

您在应用内创建的习惯、打卡记录、备注等数据：

  • 仅存储在您设备的本地存储（AsyncStorage）中
  • 不会上传至任何服务器或云端
  • 不会与任何第三方共享
  • 您可以随时在「设置 → 清除所有数据」中删除全部数据

三、权限说明

应用仅申请以下必要权限：

权限名称用途
发送通知（POST_NOTIFICATIONS）用于习惯提醒、成就通知、每周总结推送，您可随时在系统设置中关闭
精确闹钟（SCHEDULE_EXACT_ALARM）用于在指定时间准时发送习惯提醒通知

我们不申请以下权限：摄像头、麦克风、联系人、短信、位置、存储读写等。

四、第三方 SDK

应用使用以下开源框架，均不收集用户数据：

  • React Native（Facebook/Meta）- 跨平台应用框架
  • Expo - React Native 工具链
  • AsyncStorage - 本地数据存储，数据不离开设备
  • expo-notifications - 本地通知，不连接外部推送服务器

五、数据安全

由于所有数据均存储在您的设备本地，数据安全完全由您的设备安全保障（屏幕锁定、设备加密等）来提供。我们建议您：

  • 为手机设置安全的解锁密码
  • 定期备份重要数据
  • 不在公共设备上使用本应用

六、未成年人隐私

本应用不面向 14 周岁以下的未成年人。若您未满 14 周岁，请在监护人的指导下使用本应用。

七、本政策的更新

我们可能会不时更新本隐私政策。当政策发生重大变更时，我们会在应用内通过弹窗或通知告知您，并在本页面更新修订日期。建议您定期查阅本政策。
